Text

Tag Name Description
<b> Bold Sirve para poner el texto en negrita.
<i> Italic Srive para poner el texto en cursiva.
<sup> Superscript Sirve para superponer texto. Ej: 2th
<sub> Subscript Sirve para sobreponer texto. Ej: 223
<br> Line Breaks Sirve para romper una linea.
<hr> Horizontal Rules Sirve para crear una linea horizontal entre dos parrafos.
<q> Quotation Sirve para poner cualquier texto entre comillas.
<abbr> Abbreviations Sirve para abreviar cualquier texto.
<cite> Citations Indica dónde está la citación.
<address> Author Details Contiene los detalles del autor de la pagina.

Table Tags

Tag Name Description
<table> Table The Table element is used to create a table. The contents of the table are written out row by row
<thead> Table Head The headings of the table should sit inside the <thead> element
<tbody> Table Body The body of the table should sit inside the <tbody> element
<tfoot> Table Footer The footer of the table should sit inside the <tfoot> element
<tr> Table Row You indicate the start of each row. It is followed buy 1 or more <td> elements.
<td> Table Data Each cell of a table is represented using a <td> element.
<th> Table Heading Is used just like the <td> element but its purpose is to represent the heading for either a column or a row. (Si usamos <th scope="col"> pondremos el texto en negrita)

Table attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element shd span
rowspan Row Span Defines how many rows a td element shd span
width (dentro de <table>) Width How wide that table shd be.
cellpading (dentro de <table>) Cellpading To add space inside each cell of the table.
cellspacing (dentro de <table>) Cellspacing to create space between each cell of the table.
border (dentro de <table>) Border To indicate de width of the border in pixel.
bgcolor (dentro de <table>) Background color To indicate background colors of either the entire table or individual table cells.
